개인정보 처리방침

최종 수정일: 2024년 12월 14일

1. 소개

English Word 10000(이하 "앱" 또는 "당사")은 사용자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보 처리방침은 사용자가 당사의 모바일 애플리케이션을 사용할 때 수집, 사용 및 보호되는 정보에 대해 설명합니다.

2. 수집하는 정보

사용자가 앱을 사용할 때 당사와 제3자 파트너는 다음 정보를 자동으로 수집할 수 있습니다:

데이터 유형 목적 신원 연결 여부 추적 사용 여부
기기 ID (IDFA) 제3자 광고, 분석 예 예
구입 내역 앱 기능 (광고 제거) 예 아니오

광고 데이터 제3자 광고, 분석 예 예

2.2 사용자가 제공하는 정보

본 앱은 계정 생성이나 이름, 이메일, 전화번호 등의 개인정보 제공을 요구하지 않습니다.

3. 제3자 서비스

당사 앱은 다음과 같은 제3자 서비스를 사용하며, 이러한 서비스는 정보를 수집할 수 있습니다:

3.1 Google AdMob

당사는 광고 표시를 위해 Google AdMob을 사용합니다. AdMob은 다음을 포함한 맞춤형 광고 목적으로 데이터를 수집하고 사용할 수 있습니다:

  • 기기 식별자 (IDFA)
  • IP 주소
  • 사용 데이터 및 광고 상호작용

자세한 내용은 Google 개인정보처리방침을 참조하세요.

3.2 Apple StoreKit (인앱 구매)

당사는 인앱 구매를 위해 Apple의 StoreKit을 사용합니다. 구매 거래는 전적으로 Apple에서 처리합니다. 당사는 사용자의 결제 정보에 접근할 수 없습니다. 자세한 내용은 Apple 개인정보처리방침을 참조하세요.

3.3 MyMemory 번역 API

당사는 단어 번역을 위해 MyMemory API를 사용합니다. 번역을 요청하는 단어만 이 서비스로 전송됩니다. 개인정보는 공유되지 않습니다.

4. 정보 사용 방법

수집된 정보는 다음 목적으로 사용됩니다:

  • 제3자 광고: 관련 광고 표시
  • 분석: 앱 사용 현황 파악 및 서비스 개선
  • 앱 기능: 구매 후 광고 제거 등의 기능 활성화
  • 성능 개선: 버그 및 충돌 식별 및 수정

5. 데이터 추적

당사 앱은 광고 목적으로 추적을 사용합니다. 다음 데이터는 다른 회사가 소유한 앱 및 웹사이트 전반에서 추적에 사용될 수 있습니다:

  • 기기 ID (IDFA)
  • 제품 상호작용 데이터
  • 광고 데이터

기기 설정을 조정하여 광고 추적을 제한할 수 있습니다:

  • iOS: 설정 → 개인정보 보호 및 보안 → 추적 → "앱이 추적을 요청하도록 허용" 끄기

6. 데이터 저장

사용자의 어휘 학습 데이터(즐겨찾기, 학습한 단어, 퀴즈 결과, 설정)는 사용자의 기기에 로컬로 저장됩니다. 당사는 외부 서버에 사용자의 개인 데이터를 저장하지 않습니다.

7. 아동 개인정보 보호

당사 앱은 모든 연령대에 적합합니다. 당사는 13세 미만 아동의 개인정보를 고의로 수집하지 않습니다. 13세 미만 아동으로부터 정보를 수집했다고 생각되면 당사에 연락해 주십시오.

8. 사용자 권리

사용자의 위치에 따라 다음과 같은 권리를 가질 수 있습니다:

  • 데이터 접근 권리
  • 데이터 삭제 권리
  • 데이터 수집 거부 권리
  • 광고 추적 제한 권리
